# credit-card-fraud-detection
Credit Card Fraud Detection using Machine Learning and Deep Learning with Flask Deployment# Credit Card Fraud Detection System

## Overview
This project implements a Credit Card Fraud Detection System using Machine Learning and Deep Learning techniques.

## Models Used
- Random Forest
- Gradient Boosting
- Deep Learning Neural Network

## Technologies
- Python
- Flask
- Scikit-Learn
- TensorFlow/Keras
- HTML
- CSS
- JavaScript

## Results
- Random Forest AUC: 0.9277
- Gradient Boosting AUC: 0.9058
- Deep Learning AUC: 0.5000
